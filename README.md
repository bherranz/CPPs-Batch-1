# 🚀 42 C++ Modules — Batch 1

> *This project has been created as part of the 42 School curriculum.*

This repository contains my implementations of the first **five C++ modules** from the 42 Common Core: **CPP Module 00 to CPP Module 04**.

These modules introduce the fundamentals of **Object-Oriented Programming (OOP)** in C++ and progressively explore concepts such as classes, memory management, inheritance, and polymorphism.

The objective is to transition from **procedural C programming to modern C++**, focusing on encapsulation, reusable code, and object-oriented design.

---

# 📚 Modules Overview

## 🧩 CPP Module 00 — Introduction to C++

### Concepts learned
- Namespaces
- Classes and member functions
- `std::string`
- `std::cout` / `std::cin`
- Static variables
- Basic class design

### Exercises
- 📞 PhoneBook  
- 👤 Account management simulation

This module introduces the **basic syntax of C++** and the structure of simple classes.

---

## 🧠 CPP Module 01 — Memory and References

### Concepts learned
- Memory allocation (`new` / `delete`)
- References
- Pointers to members
- File streams
- Basic class relationships

### Exercises
- 🧟 Zombie Horde
- 🔁 String replacement tool
- ⚔️ Weapon / Human interaction

This module focuses on **memory management and references**, key concepts in C++.

---

## 🧱 CPP Module 02 — Ad-hoc Polymorphism

### Concepts learned
- Orthodox Canonical Form
- Operator overloading
- Fixed-point numbers
- Copy constructors
- Assignment operators

### Exercises
- 🔢 Fixed-point number class

This module introduces **operator overloading** and deeper control over object behavior.

---

## 🧬 CPP Module 03 — Inheritance

### Concepts learned
- Class inheritance
- Base and derived classes
- Constructor / destructor chaining
- Function overriding

### Exercises
- 🤖 ClapTrap
- 🛡️ ScavTrap
- 💥 FragTrap
- 💎 DiamondTrap

This module explores **inheritance and class hierarchies**.

---

## 🧠 CPP Module 04 — Subtype Polymorphism

### Concepts learned
- Subtype polymorphism
- Virtual functions
- Abstract classes
- Interfaces
- Deep vs shallow copies

### Exercises
- 🐶 Animal hierarchy
- 🧠 Brain class
- 🧪 WrongAnimal example

This module introduces **true polymorphism**, one of the most powerful features of C++.

# 🛠️ Compilation

All modules are compiled using:

```bash
c++ -Wall -Wextra -Werror -std=c++98
```
Example:
```bash
make
./program_name
```
